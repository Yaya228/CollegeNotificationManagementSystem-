# webTask

#### 介绍
web设计大作业
学生信息收集系统
JAVA web


#### 安装教程

1.  下载解压
2.  添加jar包:mysql-connection-mysql-5.1.7.jar
3.  run on servers

#### 使用说明

1.  安装mysql数据库
2.  拷贝WebContent / webdatabase.sql
3.  运行sql文件

#### 运行
1.  运行WebContent /entry /index.html
